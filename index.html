Connect to your database

Your data lives in Render's managed PostgreSQL via Supabase client — encrypted at rest, no data on-device. The setup takes under 5 minutes and you only do it once.

1
Go to render.com, create a free PostgreSQL database. Copy the External Database URL.
2
Go to supabase.com, create a project. Under Settings → Database, connect it to your Render Postgres using the external URL, or just use Supabase's built-in Postgres (simpler).
3
In Supabase SQL Editor, run the included schema.sql file to create all tables.
4
Paste your Supabase Project URL and anon key below. Credentials stay only in your browser's localStorage.
